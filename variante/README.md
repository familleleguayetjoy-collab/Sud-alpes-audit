# Variante — identité navy & or

Copie complète du site S2A avec la nouvelle identité visuelle, **sans aucune modification du site d'origine** (qui reste à la racine du dépôt).

## Ce qui change

- **Palette** : navy `#0F1E2E` / `#1E3A52` / `#415466`, or `#C8A57A`, clair `#F2F4F7` (l'accent rouge et les tons chauds de l'original sont remplacés partout)
- **Typographies** : Poppins (titres) + Montserrat (texte), auto-hébergées dans `assets/fonts/` — aucune dépendance externe
- **Logo** : wordmark « S2A » avec flèche or ascendante, recréé en SVG vectoriel inline (header + footer)
- **Favicon** : SVG navy/or assorti
- **Slogan** : « Avancer. Décider. Réussir. » dans le footer

## Notes

- Toutes les pages portent `noindex,nofollow` pour ne pas concurrencer le site principal pendant les essais. À retirer si cette variante devient le site définitif.
- Pour prévisualiser : ouvrir `variante/index.html`, ou déployer le dossier `variante/` comme site séparé sur Netlify (publish directory : `variante`).
